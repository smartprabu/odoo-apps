Lock POS Screen

Users/Cashiers can unlock POS screen by scanning their barcode or using security PIN

Version: v12.0.1.0.0


Tested and maintained by
IT Projects Labs
Assitance: help@itpp.dev

Key features:
  • Lock POS screen at any time
  • Unlock screen using barcode
  • Unlock screen using security PIN
  • Set autolock time

How It Works

Go to Settings → Users menu and open a user form. Be sure that Barcode and/or Security PIN are set in the Point of Sale tab.


Lock POS Screen

Open POS session and click on user name at the top of POS screen. Then click the logout icon to lock the screen.



Unlock POS Screen

To unlock it, you have to make one of the following:

  • scan user barcode when screen locked
  • select a user and input security PIN


Unlock by Barcode

The warning if barcode is incorrect


Unlock by PIN

The warning if PIN is incorrect


If Barcode or PIN is valid, POS is unlocked succesfully.


Screen Autolock

In POS configuration form you can set a last activity interval to lock the screen automatically.
